В папке Graphics содержатся все графики.
1) Strings (no subs) - для каждого из типов шаблонов на разных размерностях построены 3 графика времени работы алгоритвом. Binary text 10 000 все 3 алгоритма и т.д

2) Далее ещё 4 файла, в которых для каждого типа шаблонов отдельно разделены графики с подстановками.
Берём первый файл BinatySubs(10 000): тут каждый из 3 алгоритмов прогоняется на этих даннных сперва для 1 подстановки, потом для 2 и т.д

И так для кажого из 4 файлов.
