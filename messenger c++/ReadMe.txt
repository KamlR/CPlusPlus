Есть клиент и сервер, которые обмениваются друг с другом сообщениями, используя сокеты. В программе работает два процесса для обмена сообщениями. Изначально запускается сервер (server.cpp), который ждёт подключения клиента (client.cpp). Затем запускается клиент, и начинается обмен сообщениями. Чтобы завершить переписку - один из участников чата должен ввести end.
